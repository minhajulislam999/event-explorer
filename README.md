# Event Explorer

## Purpose
A local event discovery platform where users can browse, explore, and reserve seats for upcoming events like tech conferences, art exhibitions, sports tournaments, and more.

## Live URL
[Event Explorer](https://event-explore.netlify.app/login)

## Key Features
- Browse upcoming local events by category
- User authentication (Email/Password & Google Login)
- Reserve seats for events
- Update profile (name & photo)
- Forgot password with email prefill
- My Bookings page
- Fully responsive design
- Dynamic page titles
- Protected routes

## NPM Packages Used
- react-router-dom
- firebase
- react-toastify
- swiper
- daisyui
- tailwindcss
